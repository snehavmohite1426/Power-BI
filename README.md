Amazon Sales Analysis Dashboard
**Project Overview:**
This project presents an interactive Amazon Sales Analysis Dashboard built using Power BI.
The dashboard provides insights into sales performance, product trends, and category-level analysis to support data-driven business decisions.

**Business Objective:**

1.To analyze Amazon sales data and identify:

Year-to-Date (YTD) Sales performance
Top-performing product categories
Monthly and weekly sales trends
Best-selling products

2.Customer engagement based on reviews

Key KPIs
YTD Sales
YTD Products Sold
YTD Reviews
QTD Sales
Sales by Month & Week
Dashboard Features

3.Sales trend analysis (Monthly & Weekly)

Top 5 products by Sales

Top 5 products by Reviews
Category-wise sales contribution
Interactive slicers for filtering
Clean, modern dark-themed UI

**Tools & Technologies Used:**

Power BI
DAX (Data Analysis Expressions)
Data Modeling
Excel (Data Preparation)

**Key DAX Measures Used:**

YTD Sales
QTD Sales
YTD Product Sold
YTD Reviews
